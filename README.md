# Tanuki_Tachie LP

ゆっくり立ち絵素材「たぬき式ゆっくり」を配布するための静的LPです。

## ファイル構成

- `index.html`: LP本体
- `styles.css`: デザイン
- `script.js`: スクロール表示アニメーション
- `downloads/tanuki-yukkuri.zip`: 配布用ZIPの配置先

## 使い方

1. 配布ファイルを `downloads/tanuki-yukkuri.zip` に置く
2. `index.html` をブラウザで開く
3. 文言を実際の収録内容・利用規約に合わせて更新する

## 補足

- Google Fonts を参照しています。完全オフラインで使う場合は `index.html` のフォント読み込みを削除し、`styles.css` のフォント指定をローカルフォント寄りに調整してください。
- 収録内容や利用規約が未確定の箇所は仮文言です。
